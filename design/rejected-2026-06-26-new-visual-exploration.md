# Rejected Design Exploration: 2026-06-26 New Visual Directions

## Decision

This exploration is rejected and must not be used as a design source for dev or test.

## Why It Was Rejected

- It treated the homepage IA problem as a full visual redesign problem.
- It introduced new visual directions instead of staying anchored to the existing Figma Make design and v2.4 visual lock.
- It risked replacing the established Head in the Clouds / 云上心事 product language: dawn atmosphere, paper-ticket Cloud Card, Chinese serif, English italic display, and flight metadata.
- It produced a utility-app navigation direction before PM/PMO had reconciled it with the original 01a / 01b design.

## Correct Boundary

The design task is not to redesign the product.

The correct task is:

```text
Keep the existing Figma Make / v2.4 visual language.
Patch the post-onboarding homepage IA, CTA hierarchy, and repeat-visit fatigue.
Do not replace the product's visual identity.
```
